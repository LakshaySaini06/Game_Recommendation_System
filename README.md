# Game_Recommendation_System
YourNextGame is a modern, AI-powered web application that helps gamers discover their next favorite game based on mood, preferences, and personalization.  Instead of browsing endless game lists, users simply select how they feel (Action, Chill, Story, Multiplayer, etc.), and the system intelligently recommends games tailored to them.
Features
Mood-based game recommendations
Lightweight AI personalization (age, region, preferences)
Search, filter & sort games by genre, rating, and release year
Favorites system with gamer persona analysis
Modern UI/UX with glassmorphism, animations & Three.js background
Local storage (no backend required)
